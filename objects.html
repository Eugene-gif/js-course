Объекты

Структура и синтаксис:
const myCity = { city: 'New Yourk', popular: true, country: 'USA', } // где city - это свойство // 'New Yourk' - это значение свойства

Порядок свойств в объекте не имеет значения

Получение значений свойств:
const myCity = { city: 'New Yourk', popular: true, country: 'USA', } console.log(myCity.city); // 'New Yourk' console.log(myCity.popular); // true

Такой синтаксис называется точечной записью

Изменение значений свойств:
const myCity = { city: 'New Yourk', } myCity.city = 'Krasnodar'; console.log(myCity); // {city: 'Krasnodar'}

Добавление новых свойств:
const myCity = { city: 'New Yourk', } myCity.popular = true; myCity.country = 'USA'; console.log(myCity);

Удаление свойств:
const myCity = { city: 'New Yourk', popular: true, country: 'USA', } console.log(myCity); delete myCity.country; console.log(myCity);

Используем оператор удаления delete для удаления свойств

Доступ к значению свойства с использованием скобок:
const myCity = { city: 'New Yourk', } myCity['popular'] = true; console.log(myCity); // {city: 'New Yourk', popular: true} const countryPropertyName = 'country'; myCity[countryPropertyName] = 'USA'; console.log(myCity); // {city: 'New Yourk', popular: true, country: 'USA'}
Вложенные свойства:
const myCity = { city: 'New Yourk', info: { isPopular: true, country: 'USA', } } console.log(myCity.info.isPopular); // true delete myCity.info['isPopular']; console.log(myCity) // {city: 'New Yourk', info:{ country: 'USA'}}

Использование переменных:
const name = 'Eugene'; const postsQty = 23; const userProfile = { name: name, postsQty: postsQty, hasSignedAgreement: false, }

Когда имя свойства совпадает с названием переменной, то можно использовать сокращённую запись:

const name = 'Eugene'; const postsQty = 23; const userProfile = { name, postsQty, hasSignedAgreement: false, }
